The MAIN Stack Framework

Download MAIN.0.8.1.zip

ng-io.com is the home of the MAIN stack framework, which was created to make Angular and Web Sockets work together seemlessly. It revolutionizes how we do things by making the <form> tag and AJAX a thing of the past! This site, which is a complete working example of it, is here to educate you about how to do it all. Workflow & Compilation with Grunt

I am using Grunt in my workflow, which is a task runner, to perform the translation of the less styles and the consolidation of all the scripts found in the source folder so that index.html points to a single styles.css and minified script.js file. You could skip these steps and just keep everything in the www/ folders instead.
